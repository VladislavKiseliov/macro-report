# Macroeconomic Data Analyzer

Утилита для анализа макроэкономических данных из CSV-файлов. Скрипт позволяет объединять данные из нескольких источников и формировать аналитические отчеты.

## Требования

- Python 3.10+
- Зависимости из `requirements.txt` (основная — `tabulate`)

## Установка

1. Клонируйте репозиторий.
2. Создайте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/macOS
.venv\Scripts\activate     # для Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Запуск осуществляется через терминал. Скрипт принимает список файлов через флаг `--files` и тип отчета через `--report`.

Пример запуска отчета по среднему ВВП:

```bash
python main.py --files data_examples/economic1.csv data_examples/economic2.csv --report average-gdp
```

## Тестирование

Для запуска тестов используется pytest. Проверки включают валидацию файлов, корректность расчетов и обработку поврежденных данных.

```bash
python -m pytest
```

## Для разработчиков: добавление нового отчета

Архитектура скрипта позволяет легко добавлять новые типы анализа без изменения логики загрузки данных:

1. Создайте новую функцию, которая принимает `List[Dict]` и возвращает `List[Dict]`.
2. Зарегистрируйте ее в словаре `REPORT_REGISTRY` внутри функции `main()` под новым ключом.
3. Теперь отчет доступен через аргумент `--report <ваш_ключ>`.
